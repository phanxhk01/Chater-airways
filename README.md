## Usage Guidelines

**Primary colour**  
Deep maroon (#8C2F2F) — used for the wing on light or transparent backgrounds.

**Secondary colour**  
Crisp white (#FFFFFF) — for dark backgrounds or reversed treatments.

**Clear space**  
Maintain a minimum of 1× the wing’s height around the logo on all sides.

**Do not**  
- Stretch, skew, or recolour the symbol except using the approved monochrome versions  
- Place text or other elements inside the wing  
- Use low-resolution versions for print or large digital displays

**Recommended applications**  
- Website & app: PNG transparent (CTW)  
- Dark-themed platforms (GeoFS, Discord, social): LTRRD.png  
- Light backgrounds & print: LTRW.jpeg or LTRBK.jpeg  
- Merchandise & signage: CTR.jpeg (maroon field)
